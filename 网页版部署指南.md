# 🌐 虫米 App — 网页版 (Web App) 免费部署与“添加到主屏幕”指南

由于您选择了 **方案 A（网页版）**，这将是解决 iOS（苹果手机）及跨平台安装最完美的方案。
通过将 App 部署到免费的云端服务上，您和伴侣只需使用手机浏览器打开链接，将其 **“添加到主屏幕”**，即可像使用普通 App 一样在手机上拥有专属的桌面图标和全屏无边框的完美体验！

---

## 第一步：本地编译网页文件

我们已经为您在项目中初始化了 Web 支持。首先，在电脑上生成静态网页文件：
1. 打开 PowerShell 终端，进入项目根目录：
   ```powershell
   cd e:\AI-based-project\love-app
   ```
2. 运行编译命令：
   ```powershell
   $env:PATH = "E:\flutter\bin;$env:PATH"
   flutter build web --release
   ```
   *编译成功后，生成的全部网页文件都保存在以下文件夹中：*
   👉 `e:\AI-based-project\love-app\build\web\`

---

## 第二步：选择免费部署上线方案

以下为您提供两种最主流、完全免费且 10 秒就能上线的部署方案：

### 选项 1：使用 Vercel 命令行一键部署（最快，只需 1 分钟）
如果您的电脑上装有 Node.js（即可以使用 `npm` 命令）：
1. 在 PowerShell 终端中，直接进入编译好的网页目录：
   ```powershell
   cd e:\AI-based-project\love-app\build\web
   ```
2. 执行 Vercel 部署命令：
   ```powershell
   npx vercel
   ```
3. **根据控制台提示进行回车确认**：
   - 提示 `Set up and deploy ...?` 输入 `y` 并回车；
   - 提示登录，按提示在浏览器中通过 GitHub 或邮箱登录即可；
   - 提示 `Which scope ...?` 直接按回车；
   - 提示 `Link to existing project?` 输入 `N`（否）并回车；
   - 提示 `What’s your project’s name?` 直接按回车；
   - 提示 `In which directory ...` 直接按回车。
4. 仅需几秒钟，终端就会输出部署成功，并给您一个专属于您们情侣的免费网址，例如：
   👉 `https://chongmi-xxxx.vercel.app`

---

### 选项 2：通过 GitHub 自动关联部署（最推荐，未来修改代码自动更新）
1. 将您的代码上传到您个人的 **GitHub** 仓库（可以设为私有仓库保护隐私）。
2. 打开 [Vercel 官网 (vercel.com)](https://vercel.com) 注册并登录账号（推荐使用 GitHub 账号直接登录）。
3. 在 Vercel 控制台点击 **“Add New” -> “Project”**，导入您刚刚上传的 `love-app` 仓库。
4. **修改构建配置 (Build & Development Settings)**：
   - **Framework Preset**：选择 **`Other`**。
   - **Build Command**（构建命令）：开启 Override 并填写：
     `flutter build web --release`
   - **Output Directory**（输出目录）：开启 Override 并填写：
     `build/web`
5. 点击 **“Deploy”**。
   - 以后每次您只要往 GitHub 提交代码，Vercel 就会**自动在云端帮您编译并更新网页版**，手机端无需做任何操作！

## 第三步：手机端“安装”并使用 (PWA)

当您拿到部署好的专属网址（例如 `https://your-love-app.vercel.app`）后，就可以在您和伴侣的手机上“安装”了：

### 🍎 iOS (苹果手机 iPhone) 操作步骤：
1. 在 iPhone 上打开 **Safari 浏览器**，输入并访问您的专属网址。
2. 页面加载完成后，点击 Safari 底部工具栏正中间的 **“分享”** 按钮（带有向上箭头的正方形图标）。
3. 在弹出的菜单中向下滚动，找到并点击 **“添加到主屏幕” (Add to Home Screen)**。
4. 确认应用名称为 **“虫米”**，点击右上角的 **“添加”**。
5. **体验效果**：您的 iPhone 桌面上就会出现一个精致的“虫米”图标。**点开它，App 将以全屏状态启动（没有浏览器的地址栏和返回键，和原生 App 体验完全一致）**。

### 🤖 Android (安卓手机) 操作步骤：
1. 用手机浏览器（如 Chrome、系统自带浏览器等）打开您的专属网址。
2. 浏览器会自动检测到该网页支持应用化，并在底部提示：“将虫米添加到主屏幕” 或 “安装应用”；
3. 如果没有提示，可点击浏览器右上角的 “三个点” 菜单，选择 **“添加到主屏幕”** 或 **“安装 App”**。
4. 确认后，安卓桌面上就会生成“虫米”的快捷图标，点开即为全屏 App 体验。
